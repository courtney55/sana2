# README

Sana

Sana can be used to search for bars with live music in Chicago. Users can create an account and write reviews. 

Ruby version
2.5.3

Try it now

Go to https://sana2-courtney-fink.herokuapp.com/ to explore or create an account. 

Tools Used

-Bootstrap
-Devise
-Geocoder
-Figaro
-Heroku
